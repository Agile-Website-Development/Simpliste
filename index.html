Simpliste

Simpliste is a simple responsive HTML5 template which is a good start to create one page sites, simple sites or landing pages. It’s free and comes with ready to use skins.

  • en
  • ru

Standards

Use HTML5 and CSS3 with no effort modern technologies a for your service with semantic tag names and skins with no image files

One site, many devices

Responsive means you dont’t have to bother about alternative versions of your site. It can be used on desktop, tablets and smartphones

Choose your skin

Simpliste has many different skins from really simple to bizarre. They are all designed without using any image files

Simple

  1. Simpliste isn't a framework
  2. It doesn't require a CMS
  3. Download and use it on your server right away: it has prepared classes, many skins and automatically responsive markup

Usage

Simpliste has simple class names. You just use the one you need at the moment.

.col_100 .col_75 .col_66 .col_50 .col_33 .col_25
Columns

.right .left .center
Text align

.button .table .error .message .success
Buttons, tables, messages

There a more but only the necessary.

File structure

 /_www/ Main folder. Contains html-file, CSS styles and favicon

 /_xtensions/ Folder with Xtensions

 /skinname/skin.css Folders with skins, containing skin.css files

Xtensions

/_xtensions/ folder contains code snippets and plugins optimised to use with Simpliste.
